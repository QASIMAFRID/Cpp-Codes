## Getting Started

### Clone the Repository

```bash
git clone https://github.com/QASIMAFRID/Cpp-Codes.git
```

### Run a Project

Navigate to any project folder and compile the `.cpp` files using your preferred C++ compiler.

Example using g++:

```bash
g++ main.cpp -o app
./app
```
